O que é que é o cinema?

O cinema é um artefato cultural criado por determinadas culturas que nele se refletem e que, por sua vez, as afetam.
É uma arte poderosa, é a fonte de entretenimento popular e,
destinando-se a educar ou doutrinar,
pode tornar-se um método eficaz de influenciar os cidadões.
É a imagem animada que confere aos filmes o seu poder de comunicação universal.
