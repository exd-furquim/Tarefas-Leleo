  • Quem criou o JavaScript e em que ano ele foi criado

  • O JavaScript foi criado em **1995** por **Brendan Eich**, enquanto ele trabalhava na empresa **Netscape Communications Corporation**. Inicialmente, o JavaScript foi desenvolvido como uma linguagem de script para ser executada no lado do cliente (no navegador) para melhorar a interação do usuário nas páginas web. O primeiro nome do JavaScript foi **Mocha**, depois passou a ser chamado **LiveScript** e, finalmente, foi renomeado para **JavaScript**, em uma tentativa de capitalizar sobre a popularidade do Java, que estava em alta na época. No entanto, JavaScript e Java são duas linguagens bastante diferentes.

  • O que são eventos em JavaScript? Dê um exemplo simples.

  • Em JavaScript, eventos são ações ou ocorrências que acontecem no navegador, como o clique de um botão, o movimento do mouse, o envio de um formulário, ou o carregamento de uma página. Esses eventos podem ser detectados e manipulados pelo JavaScript para executar código em resposta a essas ações. Por exemplo, você pode criar um evento para responder quando um usuário clica em um botão ou quando ele passa o mouse sobre um elemento.

  • O que é Visual Studio Code?

  • O VS Code (Visual Studio Code) é um editor de código-fonte desenvolvido pela Microsoft. Ele é leve, rápido e altamente customizável, sendo uma das ferramentas mais populares entre desenvolvedores. Embora seja um editor de código, o VS Code oferece funcionalidades que o tornam bastante completo para desenvolvimento de software, como suporte para diversas linguagens de programação, depuração, controle de versão (Git), e extensões para personalizar ainda mais seu ambiente de trabalho

  • O que é Bootstrap?

  • O Bootstrap é um framework front-end de código aberto, desenvolvido inicialmente por Mark Otto e Jacob Thornton na Twitter. Ele é utilizado para desenvolver websites e aplicativos web responsivos e com uma interface visual moderna e consistente. O Bootstrap oferece uma série de ferramentas e componentes prontos, como botões, menus, cards, tabelas, formulários e muito mais, que podem ser usados para criar sites de maneira rápida e eficiente.
